# hw03 人脸检测 Web 应用
## 项目说明
- 基于 Streamlit + OpenCV 实现人脸检测，无 dlib/face_recognition 依赖
- 支持上传图片或使用示例图片，自动检测人脸并标注红色框

## 运行步骤
1. 安装依赖：
   ```bash
   pip3 install streamlit opencv-python pillow numpy
2、启动应用：
streamlit run app_new.py
3、访问链接：
http://localhost:8501
4. 点击「提交」→ 修改完成。
